getfriendgroups

Name

getfriendgroups — Retrieves a list of the user's defined groups of friends.

Mode Description

Retrieves a list of the user's defined groups of friends.

Arguments

  1. [struct](required) Containing keys:

    • username:

      • [scalar](required) Username of user logging in.

    • password:

      • [scalar](required) Password of user logging in in plaintext. Either this or "hpassword" must be present.

    • hpassword:

      • [scalar](required) MD5 digest of user's password. Not much more secure than password, but at least it's not in plain text.

    • ver:

      • [scalar](required) Protocol version supported by the client; assumed to be 0 if not specified. See Chapter 8 for details on the protocol version

Return Values

  1. [struct](required) Containing keys:

    • friendgroups:

      • [list](required) Containing items:

        • [struct](optional; multiple allowed) Containing keys:

          • id:

            • [scalar](required) The bit number for this friend group, from 1-30.

          • name:

            • [scalar](required) The name of this friend group.

          • sortorder:

            • [scalar](required) The sort integer for this friend group, from 0-255.

          • public:

            • [scalar](required) Either '0' or '1' for if this friend group is public.
