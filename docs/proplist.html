Chapter 9. Journal Item Meta-data

This is a list of the meta-data that can be assigned to journal items.

Log Prop List

current_mood

Current Mood. Your current mood.

  • Datatype. char

  • Scope. general

current_moodid

Current Mood ID#. Your current mood ID number, if known.

  • Datatype. num

  • Scope. general

current_music

Current Music. Music you're currently listening to.

  • Datatype. char

  • Scope. general

opt_preformatted

Don't Auto-Format. Turn on if post contains HTML and shouldn't be formatted

  • Datatype. bool

  • Scope. general

opt_nocomments

Don't Allow Comments. Turn on if readers can't post comments on this entry.

  • Datatype. bool

  • Scope. general

picture_keyword

Picture Keyword. A keyword that should align to a defined picture

  • Datatype. char

  • Scope. general

opt_backdated

Back-dated. Set to true if this item shouldn't show up on people's friends lists (because it occured in the past)

  • Datatype. bool

  • Scope. general

opt_noemail

Don't email comments. Turn on if the poster isn't interested in receiving comments to this post by email

  • Datatype. bool

  • Scope. general

unknown8bit

Unknown 8-bit text. True if text has 8-bit data that's not in UTF-8

  • Datatype. bool

  • Scope. general
